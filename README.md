# **Installation**

**For GraphQl Server**

1. Clone the repository. 

   `git clone git@github.com:souravlahoti/graphql-book-database.git`

2. `cd server`

3. `npm install`

4. `node index.js` to start the server

**For React FrontEnd**

1. `cd client`

2. `npm install`

3. `npm start` to start the server
